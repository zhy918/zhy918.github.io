一、HTML5 Canvas（画布） 简介

HTML5 的 canvas 元素使用 JavaScript 在网页上绘制图像。
画布是一个矩形区域，您可以控制其每一像素。
canvas 拥有多种绘制路径、矩形、圆形、字符以及添加图像的方法。

二、浏览器支持

支持 <canvas> 元素的浏览器：Internet Explorer 9+、Firefox、Opera、Chrome、Safari 。
注意: Internet Explorer 8 及更早 IE 版本的浏览器不支持 <canvas> 元素。

三、创建<canvas>标签

默认情况下 <canvas> 元素没有边框和内容，通常需要指定一个id属性 (脚本中经常引用)， width 和 height 属性定义的画布的大小，使用 style 属性来添加边框。
